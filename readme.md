## Newsapp Project 

All the latest news on your fingertips.

Stack involved is MERN.

Live at https://fullstack-newsapp.onrender.com

## Initial Setup

1. Clone the project

2. Install the packages 

    ```shell
        npm i package.json
    ```

3. Create .env for environment file

    ```shell

        sudo vi .env
    ```

    Add keys

        1. PORT
        2. CONNECTION_URI
        3. NEWSAPI_KEY

4. Run the app

    ```shell
        npm run dev
    ```
